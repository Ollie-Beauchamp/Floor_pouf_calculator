# Floor_pouf_calculator
Pattern creator for a floor pouf (floor pillow for sitting)

Floor pouf design: Will be a circular pillow segmented into round-bottomed triangles. The application here will create a single-page printout (or multiple letter-sized page printouts) pattern for a single triangle, with seam allowances.

###
Inputs will be the desired diameter of finished pillow and number of triangles to segment each face into

###
Outputs will be a single page pattern for one of the triangles, with instructions for how to sew together your pillow.
